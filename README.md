# متجر الاشتراكات الرقمية الفاخر

موقع إلكتروني فاخر لبيع الاشتراكات الرقمية وبطاقات الألعاب، مبني بـ HTML و CSS.

## الميزات

- **قسم ترحيب فاخر جداً (Hero)**: مع شارة ترحيب، عنوان متدرج، وصفين من الأزرار وإحصائيات مميزة
- **Floating Contact Menu مميز**: قائمة عائمة مع 4 أيقونات تواصل (WhatsApp، Telegram، Email، نموذج اتصال)
  - أيقوناتملونة بألوان تدرجية
  - تأثيرات hover وanimation سلسة
  - labels عند الـ hover
  - يظهر/يختفي بضغطة زر
- **إشعار ترحيبي ذكي**: ينبثق تلقائياً عند دخول المستخدم مع زر إغلاق وإغلاق تلقائي
- **ترتيب منتجات احترافي**: 3 أعمدة على الشاشات الكبيرة
- **شارات أمان وثقة**: SSL Encrypted, Google Safe, Safari Compatible, 100% Secure
- **تكامل Telegram Bot**: الـ emails تصل إلى البوت بدون تحويل المستخدم
- **متوافق تماماً مع الهواتف**: responsive design كامل
- **حماية كاملة دون تسجيل**: لا حسابات، لا بيانات شخصية
- **اشتراك في النشرة الإخبارية**: مع تصميم فاخر وتلغرام integration
- **قسم تواصل محسّن**: نموذج جميل وآمن
- **حماية أمان عالية**: CSP headers, X-Content-Type-Options
- **دعم أكثر من 500 مستخدم**: موقع ثابت عالي الأداء

## الاستخدام

1. افتح `index.html` في متصفح الويب لرؤية المتجر.
2. انتقل بين الأقسام باستخدام شريط التنقل.
3. اضغط على "اشتر الآن" لفتح واتساب برسالة جاهزة.

## ملاحظات الأمان

هذا موقع HTML/CSS ثابت. للأمان الكامل عند الاستضافة عبر الإنترنت:
- استخدم HTTPS (شهادة SSL)
- استضف على خادم آمن
- أضف رؤوس أمان المحتوى (CSP)
- نفذ مصادقة مناسبة إذا أضفت خلفية
- حدث التبعيات بانتظام (لا توجد هنا)

الموقع يحتوي على علامات أمان أساسية لمنع الثغرات الشائعة.

## استكشاف الأخطاء

- تأكد من وجود متصفح ويب حديث مع تفعيل JavaScript (لإرسال النماذج).
- إذا لم تعمل روابط واتساب، تأكد من تثبيت واتساب على جهازك.
- الصور بدائل؛ استبدلها بصور المنتجات الفعلية للإنتاج.
- لإضافة صور، أنشئ مجلد "images" وضع الصور هناك مع الأسماء المحددة في الكود.

## الأداء

- يدعم أكثر من 500 مستخدم بدون خادم، حيث هو موقع ثابت.
- تحميل الصور الكسول لتحسين السرعة.
- تصميم متجاوب لجميع الأجهزة.
